# CI/CD Pipeline using Jenkins, Docker, Kubernetes

## Project Overview
This project demonstrates a complete CI/CD pipeline using Jenkins to automate build and deployment of a Dockerized web application onto a Kubernetes cluster.

## Tools Used
Jenkins | Docker | DockerHub | Kubernetes | Git | GitHub

## Pipeline Flow
Developer → Push Code → Jenkins → Build Docker Image → Push to DockerHub → Deploy to Kubernetes

## Expected Outcome
Zero-downtime deployment with automated versioned builds and faster release cycles.
